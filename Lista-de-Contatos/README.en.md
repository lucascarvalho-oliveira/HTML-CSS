# Project Documentation: Contact List
## Overview

This repository contains the Contact **List project**, developed with `HTML` and `CSS`, with the goal of creating a visual interface to display contacts in an organized way. The project presents images, names, emails, and action buttons (edit and delete) for each contact.

The project consists of the following files:

| File / Folder | Description |
|-----------------|-----------|
| index.html      | Main page structure (HTML) |
| stayle.css      | Stylesheet (CSS) |
| Img/            | Folder containing images used on the site (contact photos and action icons) |
| README.md       | Project documentation |

## Technologies Used

- **HTML5**: Semantic structure for the contact list
- **CSS3**: Visual styling, flexible layout, and hover effects

## Project Objectives

- Apply `HTML` and `CSS` knowledge
- Create a visually organized and interactive contact list
- Demonstrate hover effects and Flexbox layout
- Practice code organization and style separation

## Features

- Contact list with photo, name, and email
- “Edit” and “Delete” action buttons that appear when hovering over an item
- Responsive layout inside a centered container
- Rounded borders and contrasting colors for visual emphasis
- Hover effect on each list item revealing action options

## Project Style

- **Color palette**: blue background, beige container, darker tones for hover effects
- **Layout**: Flexbox used to align photos, information, and buttons
- **Action buttons**: partially transparent and only visible on hover
- **Images**: circular profile pictures for contacts

## Critical Analysis

- Semantic Structure: Proper use of `<ul>` and `<li>` lists to organize contacts
- Code Organization: Clear separation between `HTML` and `CSS`
- Visual Style: The layout is clean and interactive, highlighting hover effects

## Areas for Improvement

- **Responsiveness**: Improve layout for smaller screens and mobile devices
- **Accessibility**: Add ARIA attributes and improve color contrast for readability
- **Functionality**: Make the edit and delete buttons functional using JavaScript
- **Performance**: Optimize images to reduce loading time

## Conclusion

The Contact List project serves as solid practice in `HTML` and `CSS`, demonstrating how to structure and style interactive lists. It provides a foundation for future improvements in functionality and responsiveness, while applying good interface design practices.
