# Project Documentation: landing-page-hamburgueria

## Overview
This repository contains the Burger Snack Bar project, developed with `HTML` and `CSS`, with the goal of applying fundamental concepts of these technologies and creating a visually appealing webpage. The site presents the snack bar’s most popular products, contact information, and links to social media.

The project consists of the following files:

| File / Folder | Description |
|-----------------|-----------|
| index.html      | Main page structure (HTML) |
| CSS/style.css   | Stylesheet (CSS) |
| IMG/            | Folder containing images used on the site (logo and burge |
| README.md       | Project documentation |

## Technologies Used

- **HTML5**: Semantic structure of the webpage
- **CSS3**: Visual styling of the page
- **Google Fonts**: Roboto, Roboto Condensed, and Roboto Mono
- **Material Symbols**: Icons for the user profile

## Project Objectives

- Apply basic knowledge of `HTML` and `CSS`
- Create a visually appealing and functional webpage
- Demonstrate organized code structure between HTML and CSS

## Features

- Header with logo, slogan, and sign-up/login buttons
- Top menu with links to social networks (WhatsApp, Instagram, Facebook)
- Main section displaying the most popular products with images and titles
- Navigation for Menu, Promotions, Coupons, Delivery, and Orders
- Footer with address, opening hours, and copyright information
- Buttons with hover effects and styled layout

## Conclusion

This project serves as a solid foundation in HTML and CSS, combining good practices in structure and styling. It also highlights important areas for improvement, such as responsiveness, accessibility, and optimization, which are essential for professional web development.
