# HTML-CSS

This repository gathers my **HTML and CSS study projects**, organized progressively. The goal is not just to "store code", but to **document my practical evolution in front-end development**, from basic structures to more complete and responsive layouts.

## Repository Objective

- Strengthen the fundamentals of semantic `HTML` and modern `CSS`
- Practice front-end project organization
- Improve visual hierarchy, responsiveness, and best practices
- Serve as a continuous study base (not as a final portfolio)
> Projects that become more mature and ready for professional presentation may later receive their own dedicated repositories.
